All best apps you know now in cloud

  • KVS

    KVS is Key-Value Storage Data Framework that provide high-level API for handling social data like Users, Feeds, Comments, etc. It is designed with sequential consistency in mind.

  • RIAK KV

    Riak KV is an open source Erlang application that is distributed using the riak_core Erlang library. Riak KV provides a key/value datastore interface and ready for embedding in your app.

  • MQS

    MQS is RabbitMQ client library that handles conections, channels, subscriptions has its own routing interface that you can use for building sophisticated subscriptions topology.

  • RABBITMQ

    RabbitMQ is open source message broker software that implements AMQP standard. It will powers your scalable apps and help build your own cloud.

  • N2O

    Nitrogen 2 Optimized. Binary page construction. Binary data transfer. Websocket async interface for page updates. No processes spawn, works within Cowboy processes. Page render is 2X faster than original Nitrogen.

  • Cowboy

    Cowboy is a small, fast and modular HTTP server written in Erlang. It can handle 30K req/sec on notebook. It provides complete HTTP stack in a small code base. It is optimized for low latency and low memory usage.

  • EXMPP

    This is XMPP client library for fast prototyping gateways and ejabberd supplimentary applications.

  • EJABBERD

    High-Performance Enterprise Instant Messaging Server that will power your enterprise grade chat needs.

  • GPROC

    Gproc is a process dictionary for Erlang, which provides a number of useful features beyond built-in dictionary like QLC, Global Registry.

  • SYNC

    Sync is a developer utility. It recompiles and reloads your Erlang code on-the-fly. With Sync, you can code without friction.
